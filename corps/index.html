# 防火長城

防火長城（英語：Great Firewall，常用簡稱：GFW，中文也稱中國國家防火牆，中國大陸群眾俗稱牆、網絡長城、功夫網等等），是中華人民共和國政府監控和過濾互聯網國際出口內容的軟硬件系統集合。隨著使用的拓廣，“牆”有時也被用作動詞，中國網友所說的“被牆”即指網站內容被防火長城所遮罩或者指服務器的通訊被封阻，“翻牆”也被引申為突破網絡審查瀏覽中國大陸境外被遮罩的網站或使用服務的行為。

# 金盾工程

全國警察工作資訊化工程，俗稱金盾工程，是由中華人民共和國北京政府公安機關提出的全國性網路安全基礎建設工程。 “全國警察工作資訊化工程”是中華人民共和國電子政務建設的12個重要系統建設項目之一。

# 綠壩·花季護航

綠壩·花季護航（簡稱綠壩或綠航）是中國大陸的一款用戶端的網絡過濾軟件。政府稱安裝軟件的目的在於保護未成年電腦用戶，防止他們瀏覽色情網站等有害內容，但過濾的準確度過低。另外該軟件可過濾一些政治敏感的資訊，因而有觀點認為這一軟件是政府進行網絡審查的工具之一。政府大規模推行軟件預裝措施招致諸多群眾和媒體抗告。中國大陸政府原定於2009年7月1日起，要求在中國大陸新生產和銷售的電腦進行全面預裝，後迫於輿論壓力而表示無限期延后該規定。 面對此爭議，軟件生產公司發言人則稱該強制措施並不適用於成年人；即用戶可以選擇不安裝綠壩軟件。同時官方也宣稱，上網過濾軟件可自選安裝，不會進行任何監控。
